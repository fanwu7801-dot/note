# embedded-development-note

这是一个面向嵌入式软件开发工程师成长的个人知识库，内容覆盖 STM32 裸机、RTOS、驱动开发、通信协议、文件系统、OTA、调试方法、C 语言面向对象设计、汽车电子诊断，以及架构驱动和 AI/TDD 工程化实践。

它不是单一项目的说明文档，而是一套从“基础能力”到“系统架构”的学习与设计笔记集合。推荐按“学习路径 -> 工程规范 -> 裸机/RTOS/外设 -> 驱动抽象 -> 系统架构 -> AI 工程化”的顺序阅读。

## 仓库适合谁

- 想系统补齐 STM32、RTOS、通信协议和常见外设基础的学习者
- 正在从“能写功能”走向“能做分层、抽象、架构设计”的嵌入式工程师
- 希望把 C 语言面向对象、设计模式、驱动分层和工程规范结合起来的人
- 想理解 VCU、CAN、UDS、OTA、BLE/NFC 等汽车电子方向内容的人
- 想把 AI 辅助开发从问答升级为需求约束、设计工件和 TDD 流程的人

## 说明

- 仓库以 Markdown 笔记为主，部分目录包含图片、PDF、drawio 图和示例 C 代码。
- 图片文件通常服务于同目录下的 Markdown 文档，阅读时建议使用支持本地图片预览的 Markdown 编辑器。
- 部分内容是学习过程中的阶段性笔记，建议以“理解设计思路和工程取舍”为主，不要直接把示例代码当成量产代码使用。
