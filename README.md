# embedded-development-note

这是一个面向嵌入式软件工程师成长的知识库，内容覆盖 STM32 裸机、RTOS、驱动开发、通信协议、文件系统、OTA、调试方法、C 语言面向对象设计，以及 AI/TDD 驱动的工程化思路。

它不是单一项目文档，而是一套从入门到架构实践的学习与设计笔记集合。阅读方式建议按“学习路径 -> 基础能力 -> 驱动与系统 -> 架构与工程化”逐步推进。

## 仓库适合谁

- 想系统补齐 STM32 与嵌入式软件基础的初学者
- 正在从“会写功能”走向“会做架构、会做抽象”的嵌入式工程师
- 希望把设计模式、驱动分层、调试方法和 AI 辅助开发结合起来的人
